# EV Scrollytelling Spec

## Product goal
Create an editorial landing page about electric vehicles that combines scrollytelling, interactive user controls, and strong visual hierarchy.

## Experience principles
- Use Bauhaus-inspired geometric color blocks with Swiss International typography and grid structure.
- Keep the reading flow calm, spacious, and highly legible.
- Make interactions purposeful: navigation, lens switching, and section focus should all help readers understand the story.
- Prioritize responsive behavior and accessible contrast/focus states.

## Narrative structure
1. Market shift: EVs becoming mainstream.
2. Battery intelligence: trust, range, and thermal systems.
3. Charging culture: infrastructure as user experience.
4. Cabin future: comfort, quiet, and interface design.

## Interaction model
- Sticky visual rail with scroll-aware progress.
- Active chapter state driven by section visibility.
- Clickable section navigator for direct jumps.
- Reader lens selector to highlight different EV priorities.
- Metric-based comparison studio for exploring EV trade-offs.
- End-of-story recap cards that also navigate back into the story.

## Acceptance criteria
- The home page reads as a polished editorial feature, not a starter template.
- All provided EV images are integrated into the story.
- The UI clearly reflects Bauhaus + Swiss design influence.
- Readers can interact with the page without confusion.
- The app builds successfully for deployment.
